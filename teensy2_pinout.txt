Using ATmega 32U4 controller:

Pins: row # : wire color
========================
F0 : r0 : wht
F1 : r1 : yel
F4 : r2 : red
F5 : r3 : grn
F6 : r4 : blu

Pins: column # : wire color
===========================
B7 : c0 : wht
D0 : c1 : yel
D1 : c2 : red
D2 : c3 : grn
D3 : c4 : blu
C6 : c5 : blk
C7 : c6 : blk
D7 : c7 : blu
B4 : c8 : grn
B5 : c9 : red
B6 : c10 : yel
F7 : c11 : wht
